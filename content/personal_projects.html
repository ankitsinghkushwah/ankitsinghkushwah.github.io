Eklavya Engine

A C++ game engine built with OpenGL 4.1 for Windows and MacOS.

GitHub Repo: ▶ Eklavya Engine on GitHub

Demos

Click on thumbnails below to play the video.

Features

  • Scene Management
    • Scene Graph with Frustum Culling
  • Rendering
    • Support for multiple Render Passes
    • Shadows and Post Processing
  • Animation
    • Skeletal Animation with smooth transitions
    • Inverse Kinematics
  • Physics Engine
    • Custom physics engine
    • Collision detection using Separating Axis Theorem (SAT)
    • Motion integration using Euler Integration
    • Collision resolution using impulse based methods
    • Support for Sphere and Oriented Bounding Box (OBB) collisions
    • Ray Casting and Sphere Casting
  • Asset Management
    • Texture and shader loading
    • Model and animation support via Assimp
  • Additional Tech
    • SFML Audio Library for sound effects and music
    • CMake support for Visual Studio and XCode project generation

A* Pathfinding in JavaScript

Shikhar (The Peak) For Android

I solo developed this action adventure game for Android using the Unity Engine. Inspired by Prince of Persia, I focused on creating an adrenaline rush through time based abilities, parkour, and challenging obstacles.

Gameplay Demo

Features

  • Implemented a time rewind system using Unity’s Job System
  • Parkour mechanics including ledge climbing, vaulting, wall running, sliding, and double jumping
  • Designed several levels with a focus on obstacle placement and "negative space"
  • Authored custom VFX using the Shader Graph editor
  • Built the full UI flow and player progression systems

Features
