Temple Run 2

I was fortunate enough to be part of a small team to port Temple run 2 to HTML 5 for Imangi Studios.

You can play the game here : ▶ Play Temple Run 2: Spooky Summit on Poki

Gameplay

My Responsibilities

  • Implemented Follow Camera, Character Movement & Animation State Machine.
  • Implemented Coin Collection: Converted 3D coin location to 2D Screen coordinates.
  • Implemented abilities Coin Magnet, Speed Boost, Invincible Mode & Water Surfing.
  • Graphics & Optimizations:
    • Reduced draw calls by leveraging hardware instancing wherever possible.
    • Wrote efficient shader for distanced fog effect to hide far plane culling.
    • Wrote shader to read baked lighting data.
    • Modified track pieces to be more processing friendly.
  • Added new Maps for Holidays & Festivals.
  • Level generation improvements & bug fixes.
  • Abilities Upgrades & UI
  • Saving/Loading Progress
  • Created VFX for abilities & level pieces.
